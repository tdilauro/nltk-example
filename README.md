### Setup

Required packages
- ijson (2.3)
- nltk (3.3)
- numpy (1.14.3)

These NTLK resources are needed (see [NLTK documentation](https://www.nltk.org/data.html)):
- nltk.download('punkt')
- nltk.download('averaged_perceptron_tagger')
- nltk.download('maxent_ne_chunker')
- nltk.download('words')
